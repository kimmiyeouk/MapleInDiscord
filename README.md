# MapleInDiscord

Kimmiyeouk의 개인 오픈소스 프로젝트입니다.
아무나 프로그램을 받아 수정하고 직접 사용하셔도 됩니다

* ./modules/saves/
 이곳에 유저 세이브가 들어옵니다. 여기서 수정하시면 됩니다.

* ./modules/saves/config
 디스코드 봇의 토큰을 입력해주시면 됩니다.
